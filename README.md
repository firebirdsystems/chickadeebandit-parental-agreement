# Parental Agreement

A [Chickadee Bandit](http://chickadeebandit.com) app.

Kids submit permission requests — staying out late, attending events, overnight trips. Parents are notified and must approve. Any edit resets approval and re-notifies parents.

## Features

- Kids submit requests with event details, date, and time
- Parents receive an alert and must explicitly approve or reject
- Any edit to an approved request resets it and triggers a new notification
- Full history of approved, rejected, and pending requests

## Install

In your hub, go to **Apps → Install from URL** and paste:

```
https://github.com/firebirdsystems/chickadeebandit-parental-agreement/releases/latest/download/bundle.json
```
